# Adv. Rama Goswami & Associates Website

This folder contains a professional static website for Adv. Rama Goswami & Associates.

## Quick Launch
Double-click the file below to start a local preview server:
- launch.bat

Then open:
- http://127.0.0.1:8000/

## Files
- index.html: main website page
- style.css: styling
- script.js: form and footer behavior
- robots.txt: search engine crawl instructions
- sitemap.xml: sitemap for indexing
